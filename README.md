# Techfest IIT Bombay Parallax Webpage

A visually engaging, parallax scrolling webpage themed for IIT Bombay Technology Fest (Techfest). Built with vanilla HTML, CSS, and JavaScript for a smooth, lightweight experience.

## Features
- **Parallax Scrolling:** Multi-layered sections where background, midground, and foreground elements move at different speeds.
- **Neon Dark Mode Theme:** A modern, futuristic aesthetic using cyan and magenta neon accents.
- **Smooth Animation:** Utilizes `requestAnimationFrame` for a 60fps jitter-free scroll interpolation.

## Getting Started

Simply open `index.html` in any modern web browser to view the webpage. No build tools or dependencies are required.

## Structure
- `index.html`: Contains the structural markup and layer definitions.
- `style.css`: Provides the styling, layout, fonts, and neon visual effects.
- `main.js`: Drives the interactive parallax scrolling logic.

## Assets
The `assets` folder is meant to contain images (like `logo.png`). Ensure you place the Techfest logo there to have it appear correctly in the navigation bar.
